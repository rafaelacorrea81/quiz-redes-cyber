# 🛡️ Quiz Segurança de Redes – Desafio Técnico

Projeto educacional gamificado para revisão dos conteúdos da disciplina **Segurança de Redes** no Curso Técnico em Redes de Computadores.

## 🔗 Link do Projeto

**[https://rafaelacorrea81.github.io/quiz-redes-cyber/](https://rafaelacorrea81.github.io/quiz-redes-cyber/)**

## 🎯 Quizzes Disponíveis

| Quiz | Dificuldade | Tempo/Pergunta | Pontos |
|------|-------------|----------------|--------|
| 🟢 Fundamentos de Segurança de Redes | Fácil | 45 segundos | 10 + bônus |
| 🟡 Análise de Riscos e Cenários | Médio | 2 minutos | 20 + bônus |
| 🔴 Desafio Cyber Avançado | Difícil | 2:30 minutos | 30 + bônus |

Cada quiz possui **15 perguntas** com explicações pedagógicas.

## ✨ Funcionalidades

- 3 quizzes independentes com identidade visual própria
- Temporizador por pergunta com alertas visuais
- Sistema de pontuação com bônus por velocidade
- Ranking online em tempo real (Firebase Firestore)
- Medalhas de desempenho (🛡️ 🔐 🕵️ 🚀)
- Perguntas contextualizadas e explicações didáticas
- Política de Privacidade e conformidade com LGPD
- Aviso de cookies/armazenamento local
- Regras de segurança Firestore
- Interface responsiva
- GitHub Pages

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (ES Modules)
- Firebase Firestore
- GitHub Pages

## 🔒 Segurança e LGPD

- Sanitização de nickname (remoção de HTML/scripts)
- Uso de `textContent` para dados do usuário
- Anti-spam com debounce
- Regras de segurança do Firestore
- Não coleta nome completo, CPF, e-mail, telefone, endereço, IP ou localização
